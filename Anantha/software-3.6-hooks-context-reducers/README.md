## Lesson Objectives

Learners will understand:
- How to reduce code complexity and repetition in React projects
- How to use ***reducer functions*** to manage complex state logic
- What is prop-drilling
- How to use **context** to share global storage accross components

Learners will be able to:
- Create reducer functions and apply with `useReducer` hook
- Create context providers and apply with `useContext` hook
